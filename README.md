# advanccleaning

**Next.js redesign** of [Advanced Cleaning and Restoration Techs](https://advancedcleaningandrestorationtechs.com/).

Professional water damage restoration, carpet cleaning, air duct cleaning, mold remediation, and upholstery cleaning serving Aiken SC, Augusta GA, Columbia SC and the CSRA.

## Pages Scaffolded

This repo currently contains a scaffolded Next.js site with the following pages matching the original site structure:

- `/` — Homepage (hero, why us, service areas, services, about, testimonials, CTA)
- `/services-we-offer` — Full list of services
- `/areas-we-serve` — Service area locations
- `/learn-more` — About the company
- `/contact-us` — Contact form + phone details

## Tech Stack

- Next.js 16 + App Router
- TypeScript
- Tailwind CSS
- React 19

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
